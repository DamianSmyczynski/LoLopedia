# LoLopedia
Demo: https://lolopedia.mat-kor.pl/
## Tech Stack
  
- **Backend**: 
  - Node.js
  - Nest.js
  - TypeScript
  
- **Database**: 
  - MySQL
  - TypeORM
  
- **Cache**: 
  - Redis

## Description
LoLopedia is a comprehensive web application designed for League of Legends enthusiasts, providing detailed information about champions, items, and game statistics.
